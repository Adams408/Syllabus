Course Pre-requisites: COMP 280

Course Description: : This course introduces students to programming for the World Wide Web. Topics include HTML, CSS, and client- and server-side scripting (the JavaScript and PHP languages).

Student Learning Objectives/Outcomes

Students will learn to:

  • Code HTML, CSS, JavaScript (including jQuery), and PHP, and understand how human needs
    (searching, retrieving, linking, browsing, navigating) are supported
  • Understand common web softare stacks, and their benefits.
  • Use client-server computing (and threads) on the Web and understand the HTTP protocol
  • Use Web access to databases with PHP, handle structured and stream content, and augment this with navigating links
  • Use several enhancing technologies, including JSON, AJAX, and cookies and web storage
  • On completing this course, the student should be able to produce professional Web pages and basic Web sites. The course does not cover Web design from an aesthetic or ergonomic angle, systems that facilitate Web site construction, or languages that simplify server-side scripting. The focus is on standards and scripting languages.
